Some Interesting Math Knowledge

A counter-intuitive probability problem

Assume you are worried that you might have a rare disease and thus you decide to get tested. Suppose that the testing methods for this disease are correct 99% of the time and that only 1 out of every 10,000 people in the general population has this disease. If you have been tested positive, what is the probability that you actually have this disease?

In fact, there is only less than 1% chance that you have this disease. The basic reason of this result is that this disease is so rare that you are more likely to get a false positive than actually have this disease. If you want to compute this probability using Math, you will need to learn Bayes' Theorem.

A continuous but nowhere differentiable function

The Weierstrass function is a classic example of real-valued function that is continuous everywhere but differentiable nowhere. It is named after Karl Weierstrass.

There are many obvious examples of real-valued functions that are continuous everywhere but not differentiable at some points. For instance, the absolute value function f(x) = |x| is not differentiable at the point x = 0. However, the Weierstrass function is not differentiable at any points while it is still continuous everywhere.
